# ohjelmoinnin-perusteet-harjoitusty-2025-ryhm-r21
Tämän projektin idea oli tehdä jonkin muotoinen kauppasovellus, josta ostetaan eri generoiduilta “pankkitileiltä”, jossa käyttäjä pystyy lisäämään tuotteita ostoskoriin, kun ostokset on tehty, käytetty raha poistuu tililtä. Aluksi tehtiin esimerkki konsoliin, miltä se voisi näyttää, mutta kun lähdettiin työstämään sitä GUI, huomattiin, että vaikeustaso kasvoi huomattavasti, joten luovuimme generoiduista pankkitilitoiminnosta. Sen jälkeen alettiin lisätä pikkuhiljaa uusia tiedostoja. Kaikista vaikein oli dropdownmenu.py-tiedosto, jossa piti yhdistää eri osia ja saada koko homma toimimaan yhdessä.

Sovellus toimii, kun ajetaan main.py. Tämän jälkeen käyttöliittymä avautuu ja käyttäjä voi valita tuotteen sekä määrän. Tuotteet valitaan “dropdown”-valikosta, jossa näkyy tuotteen nimi ja kappalehinta. Kun tuote on valittuna, syötetään vielä haluttu määrä tekstiruutuun jonka jälkeen painetaan “Lisää ostoskoriin”-nappia, jolloin tuote siirtyy alla olevaan ostoskoriin ja yhteenlaskettu hinta.  Kun painaa Osta nappia, ohjelma vähentää tuotteen hinnan automaattisesti pankkitililtä.  Kun “Osta”-nappia painaa ostoskorin sisältö tallentuu kuittina tekstitiedostona kuitit-kansioon. Kuitilta näkyy mm. ostaja, yhteenlaskettu hinta, ja tilillä jäljellä oleva saldo. Jos tilin saldo loppuu, käyttäjä voi tallettaa lisää rahaa 100 € kerrallaan ja rajattomasti.
